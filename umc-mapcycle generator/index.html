ckSurf UMC Mapcycle Generator

The purpose of this tool is to automatically create a UMC mapcycle template for surf servers using ckSurf.
The result will only be a template, as fine tuning for your preference has to be done manually.

Copy and paste your mapcycle.txt here, to generate a UMC Mapcycle file with tier, bonus and stage information. This enables more map information to be shown on the map vote:
